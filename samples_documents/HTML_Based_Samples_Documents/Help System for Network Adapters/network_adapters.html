Introduction

This help system provides comprehensive information for installing, configuring, and troubleshooting your network adapter. The information in this guide applies to the following network adapter models:

  • FastEthernet PC3000
  • GigaLink GL5000
  • WirelessPro WP2000
  • DualBand DBX1000

The network adapter drivers are compatible with the following operating systems:

  • Windows XP
  • Windows 2000
  • Windows Server 2003
  • Linux Kernel 2.4.x and higher

Driver Installation

Note: Before beginning installation, ensure you have administrative privileges on your computer.

Windows Installation

To install the network adapter driver on Windows:

  1. Insert the driver CD into your CD-ROM drive.
  2. If AutoPlay is enabled, the installation program will start automatically. If not, navigate to the CD-ROM drive and double-click on setup.exe.
  3. Follow the on-screen instructions to complete the installation.
  4. When prompted, connect the network adapter to your computer.
  5. Restart your computer when prompted to complete the installation.

Linux Installation

To install the network adapter driver on Linux:

  1. Extract the driver package to a directory on your hard drive: 
    tar -xzvf netadapter-linux-3.2.tar.gz
  2. Navigate to the extracted directory: 
    cd netadapter-linux-3.2
  3. Compile the driver: 
    make
  4. Install the driver: 
    sudo make install
  5. Load the driver module: 
    sudo modprobe netadapter

Warning: Do not disconnect the network adapter during driver installation. Doing so may cause the installation to fail and could potentially damage your adapter.

Network Configuration

Basic Network Settings

After installing the driver, you need to configure your network settings:

To configure basic network settings in Windows:

  1. Open the Control Panel.
  2. Click on Network Connections.
  3. Right-click on the network adapter and select Properties.
  4. Select Internet Protocol (TCP/IP) and click Properties.
  5. Configure your IP address settings as appropriate for your network.
  6. Click OK to save your settings.

Advanced Network Settings

The network adapter driver provides several advanced settings that can be configured to optimize performance:

  • Speed & Duplex: Configure the connection speed and duplex mode.
  • Wake on LAN: Enable or disable the ability to power on the computer remotely.
  • Flow Control: Enable or disable packet flow control.
  • Jumbo Frames: Configure support for jumbo frames to improve performance on gigabit networks.

Troubleshooting

Common Issues

No network connectivity:

  1. Check that the network cable is securely connected to both the adapter and the network port.
  2. Verify that the link light on the adapter is illuminated.
  3. Ensure the driver is properly installed by checking Device Manager.
  4. Try using a different network cable or port.
  5. Restart your computer.

Slow network performance:

  1. Check that you are using the latest driver version.
  2. Verify that the Speed & Duplex settings match your network infrastructure.
  3. Disable any unnecessary protocols in the adapter properties.
  4. Check for interference if using a wireless adapter.
  5. Run network diagnostics to identify bottlenecks.

Diagnostic Tools

The adapter software includes several diagnostic tools to help identify and resolve issues:

  • Link Test: Tests the connection between your adapter and the network.
  • Packet Monitor: Monitors packet transmission and reception.
  • Hardware Diagnostics: Tests the hardware functionality of the adapter.
  • Driver Verifier: Verifies the proper operation of the driver.

Advanced Settings

Performance Optimization

To optimize your network adapter performance:

  • Update to the latest driver version.
  • Configure the adapter to match your network speed (10/100/1000 Mbps).
  • Enable full-duplex mode if supported by your network.
  • Adjust the transmit and receive buffers for high-performance applications.
  • Enable offload features (TCP checksum, large send offload) when available.

Multiple Adapter Configuration

When using multiple network adapters in a single system:

  • Configure each adapter with a unique IP address on a separate subnet.
  • Use Windows advanced routing to control traffic between adapters.
  • Consider teaming compatible adapters for fault tolerance or load balancing.
